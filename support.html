ClipDock Support

Help for the Microsoft Store version of ClipDock on Windows 10 and Windows 11.

Contact support

Email limingjin777@gmail.com or use the ClipDock feedback form.

Please include the app version, Windows version, steps to reproduce and exact error text. Do not attach private screenshots or OCR text unless necessary and reviewed or redacted.

Local data and recovery

ClipDock stores its library locally by default. In Settings → Storage you can inspect workspace locations, create a verified manual backup, preview a restore and rebuild the local search index. Rebuilding the index is a repair action and does not require Pro.

OCR troubleshooting

OCR runs locally using Windows OCR components. Recognition depends on image clarity and the OCR language components installed in Windows. Check OCR status in the app, install the relevant Windows language component, then run recognition again on the selected image.

  • Prefer sharp screenshots with clear contrast and readable text.
  • Try a shorter, distinctive search term if a full sentence does not match.
  • Use a name, tag or note for critical context that OCR may not capture.

For the full workflow and limitations, read Search text inside screenshots on Windows.

Microsoft Store purchase and subscription

Trial availability, localized price, billing, renewal and cancellation are managed by Microsoft Store. Use Settings → Pro → Refresh entitlements after signing in with the purchasing Microsoft account. Subscription management opens your Microsoft account services page.

Existing screenshots, manual backup, restore and basic export remain accessible if Pro expires. Existing Lifetime owners retain their permanent Pro entitlement.

Diagnostics

Use Settings → About → Diagnostics to refresh entitlements, open local logs or copy a diagnostic summary. Diagnostics do not change your entitlement. Review copied text before sending it to support.
