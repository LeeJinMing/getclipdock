ClipDock Support

Help for the Microsoft Store version of ClipDock for Windows 10 and Windows 11.

Contact

Email limingjin777@gmail.com or use the ClipDock feedback form.

Please include the app version, Windows version, steps to reproduce and the exact error message. Do not attach private screenshots or OCR text unless they are necessary and you have reviewed or redacted them.

Local data and recovery

ClipDock stores its library locally by default. In Settings → Storage you can inspect workspace locations, create a verified manual backup, preview a restore and rebuild the local search index. Rebuilding the index is a repair action and does not require Pro.

OCR troubleshooting

OCR runs locally using Windows OCR components. Recognition depends on image clarity and the OCR language packs installed in Windows. Check OCR status in the app, install the relevant Windows language component, then retry recognition on the selected image.

Microsoft Store purchase and subscription

Trial availability, localized price, billing, renewal and cancellation are managed by Microsoft Store. Use Settings → Pro → Refresh entitlements after signing in with the purchasing Microsoft account. Subscription management opens your Microsoft account services page. Existing screenshots, manual backup, restore and export remain accessible if Pro expires.

Diagnostics

Use Settings → About → Diagnostics to refresh entitlements, open local logs or copy a diagnostic summary. Diagnostics do not change your entitlement. Review copied text before sending it to support.
